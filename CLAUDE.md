# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What this repo is

This is the GitHub special "profile README" repo (`nekosaif/nekosaif`). Its sole purpose is to render `README.md` on `github.com/nekosaif`. There is no source code, no build, no tests, and no CI — the only artifact that matters is `README.md`. Edits go live as soon as they're pushed to `main`.

## Working in this repo

- Treat `README.md` as the deliverable. Preview changes locally (any Markdown renderer) before pushing — there is no staging environment.
- GitHub renders this with GitHub-Flavored Markdown. Inline HTML is allowed but kept minimal; prefer plain Markdown to stay readable in source.
- External badges/images (shields.io, komarev, github-readme-stats) load live on every profile view. When adding one, sanity-check the URL renders.
- Project links in the "Selected Projects" section point to other repos owned by the user. When updating, verify the target repo still exists and is public.
